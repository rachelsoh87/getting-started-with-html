How the Web works

The World Wide Web (the Web) is a collection of HTML documents (web pages) that have been formatted and annotated using Hypertext Markup Language (HTML). Web pages may contain text and links to images, videos and various other types of multimedia content.

People access the Web through the internet—the world’s largest computer network where computers communicate with one another.

The computers that communicate using the internet are servers and our laptops (or desktops, cellphones, tablets); servers host the files that make up the Web, whereas our laptops use browsers to display files found on the Web.

When we want to open a link like www.wikipedia.org, our laptops send Hyper Text Transfer Protocol (http) requests to servers, the servers look up the requested HTML document (web page) and sends it back to the user’s laptop that uses browsers to displays the internet files.

Hypertext Markup Language (HTML)

HTML is used to form web pages. Web pages consist of text and other content—and markup language that determine how the text and content are displayed.

HTML elements is the structure of how tags must be placed for the document to be decipherable by web browsers.

Tags usually have an opening tag and a closing tag. If you don’t close your tags, everything after your opening tag will continue the command of your opening tag. To avoid errors, you should make your closing tag as soon as you make your opening tag—unless it is a void tag. Void tags are tags that don’t have content, such as images. Void tags do not have a closing tag.

Elements

So far, we have learnt that elements are usually either inline elements or block elements. Block elements form an invisible box, of a specific width and height, around its content.
